Documentation for the RoyalSlider Constructor Function

Fields:

  • className (type: String, default: '.js--royal-slider'): This is the CSS selector used to identify the slider element on the page. You can change this value if your slider has a different class.
  • intervalPlayTime (type: Number, default: 2500): This is the time interval (in milliseconds) after which the slider automatically moves to the next slide. You can change this value to control the speed of the slider's automatic transition.
  • progressType (type: String, default: 'basic'): This is the type of progress indicator for the slider. Currently, two types are supported: 'basic' and 'dots'. 'dots' add functionality move to N slide.
  • leftDirection (type: Boolean, default: false): This is the direction of the slider's automatic transition. If this value is set to true, the slider will automatically move to the left. If it is set to false, the slider will automatically move to the right.
